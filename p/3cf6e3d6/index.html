动作和设置

发表于2022-01-18|ps

导入

导入脚本

版本号

C:\Program Files\Adobe\Adobe Photoshop 2021\Presets/Scripts

Command + Shift + G 版本号

/Applications/Adobe Photoshop 2021/Presets/Scripts

导入动作

窗口 → 动作 → … → 载入动作 .atn

用时换成 按钮模式 会更方便

脚本

exportJPG

为当前打开文档导出 jpg，只有导单张时用

  1. 将脚本 @exportJPG.jsx 录为动作
  2. 点动作执行

像素长宽比

  1. 文件 → 脚本 → 浏览 → @像素长宽比.jsx
  2. 选择含有要执行的文件夹

install

安装脚本

  1. 文件 → 脚本 → 浏览 → @install.jsx
  2. 选择含有要安装的脚本的文件夹
  3. 重启 ps

动作

一键涂白

跟着 b 站教程做的

  1. 魔棒工具调合适的容差 32
  2. 点闭合的气泡
  3. 点动作

渐变对话框

网上下载来只用过一次但很好用，记一下过程

  1. 魔棒工具选取对话框内的文字外围，渐变1

  2. 在新出来的图层上用魔棒选取文字内围，渐变2

  3. Ctrl + E 向下合并图层

文本居中

文本图层太散列，模糊定位时用

  1. 选中图层 1 → 魔棒工具点文本框
  2. 选择工具v → 选文字图层

批处理

批量导出 JPG

文件 → 脚本 → 图像处理器

无损拼图

文件 → 脚本 → 将文件载入堆栈

尝试自动对齐源图像

载入图层后创建智能对象（智能对象 = 无损缩小）

内置批处理

配合上面的脚本和动作用

文件 → 自动 → 批处理

  • 相当于执行完后保存关闭
  • 适合已经生成为 psd 的文件（psd → psd）

  • jpg → psd 时，用上面的方法每一张都会弹窗保存
  • 在脚本或动作里添加存储行为，就不会弹窗了

覆盖动作中的”存储为“命令

首选项设置

工具 → 不勾选用滚轮缩放

文件处理 → 自动存储 5 分钟

性能 → 内存可以调大点，历史记录状态 200

暂存盘 mac 的话不用管，windows 别放系统盘就行

快捷键

编辑 → 键盘快捷键

F1 → 脚本 - @换行分层.jsx

F2、F7 → 任意动作

F11 → 动作 - 一键涂白

F3 → 图像 - 图像旋转 - 任意角度

F4 → 图像 - 调整 - 色阶（配合快速选择工具w涂掉四边方便）

F5 → 编辑 - 填充（用于内容识别）

Shift + F5 → 内容识别填充

F6 → 图像 - 调整 - 曲线

F10 @选择文本图层.jsx

鼠标侧键

修图时改笔刷大小 → [ 和 ] （这个其实一般直接用键盘）

嵌字时改文字大小 → Ctrl + Shift + , 和 Ctrl +Shift + .

中键点按 → Ctrl + 回车
